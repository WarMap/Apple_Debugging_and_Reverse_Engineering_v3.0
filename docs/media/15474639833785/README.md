# 开源

[iOSCSKit](https://github.com/droison/iOSCSKit)是微信开源的组件化框架，部分头文件保持高度一致，该博主以前在微信工作，Android端及iOS端的通信框架保持了很高程度的一致性。现阶段微信iOS端的组件化进度并未和Android端对齐（约处于Android端v2.x时代），相对落后于开源版本，当前开源版本的架构与Android端现行架构保持一致(v3.x)（[微信Android模块化架构重构实践](https://mp.weixin.qq.com/s/6Q818XA5FaHd7jJMFBG60w)）。
